# Employment Level Analysis (2015-2025)

#### QCEW Employment Growth Analysis (2015 - 2025)
Using the provided Quarterly Census of Employment and Wages (QCEW) from the Bureau of Labor Statistics (BLS), this analysis examines year-over-year changes in employment level across industries from 2015-2025. This period includes the Covid-19 pandemic, allowing us to better understand how economic shocks impact employment trends. 

#### Data Used:
* CSV quarterly single files for:
    * 2015
    * 2017
    * 2019
    * 2021
    * 2023
    * 2025 
* The primary variable visualized is: 
    * Employment level as a percent change between the third month of Q1 
        * Employment level is defined by BLS as the count of filled jobs, measured by place of work 

* Employment growth trends are visualized for the following industries: 
     * Manufacturing 
     * Trade, Transportation, and Utilities
     * Education and Health Services 
     * Leisure and Hospitality 
     * Professional and Business Services
     * Public Administration 

#### Installation 
This python script was  written and executed using Python 3.13.1 

The following libraries are required:
* pandas os numpy matplotlib 

#### How to use this code:

1. Make sure that all 6 csv files are downloaded 
2. Update the os file path to where the files are stored on your local device 
3. Run the script 
4. The script will generate a line chart showing employment growth trends across the specified industries
